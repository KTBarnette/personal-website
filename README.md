# Kyle Barnette Portfolio

Personal portfolio built with Next.js + Tailwind CSS.

## Start

```bash
npm install
npm run dev
```

## Structure

- `app/`: pages and routes
- `components/`: reusable UI sections
- `data/`: profile, project, and content data
- `public/`: static assets (resume PDF)

## Notes

- Replace placeholder profile links in `data/profile.ts`.
- Replace `public/Kyle-Barnette-Resume.pdf` with your real resume PDF.
